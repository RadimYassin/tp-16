# Application Bancaire React GraphQL

Application de gestion de comptes bancaires utilisant React et Apollo Client pour communiquer avec une API GraphQL.

## Fonctionnalités

- Création de comptes (Courant / Épargne)
- Affichage de la liste des comptes
- Transactions (Dépôt / Retrait)
- Historique des transactions

## Installation

```bash
npm install
```

## Démarrage

```bash
npm start
```

## Configuration

L'application utilise un proxy vers le backend GraphQL sur le port 8082.
